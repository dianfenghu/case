注意事项

  • 1.选用型号时请确保机械手工作全过程的力矩和转矩不超过工具快换装置的容许；

    2.选型时应考虑机器人带动末端有效荷载运动时对本产品产生的加速度力矩；

    3.领航工具快换装置具备气体压力丢失保护功能，内含气动锁紧机构，

    如果调试、应用过程中出现气源中断，工具快换装置的机器人侧和工具侧会始终保持锁紧。
